# 🌡️ Smart Temperature-Controlled Fan Using Arduino

## Overview

A smart cooling system that automatically adjusts fan speed based on ambient temperature using an Arduino Uno and TMP36 temperature sensor.

## Features

- Real-time temperature monitoring
- Automatic fan speed control
- PWM motor control
- Serial Monitor output
- LED status indication

## Components Used

- Arduino Uno R3
- TMP36 Temperature Sensor
- L293D Motor Driver IC
- DC Motor (Fan)
- Green LED
- 220Ω Resistor
- Breadboard
- Jumper Wires
- 9V Battery

## Working Principle

| Temperature | Fan Status |
|------------|------------|
| Below 25°C | OFF |
| 25°C - 35°C | Variable Speed |
| Above 35°C | Maximum Speed |
